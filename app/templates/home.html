Live Wall Street ownership data for AI agents

{{ subtitle }}

OwnerWire turns messy SEC ownership filings into clean JSON. The launch wedge is narrow on purpose: executed insider transactions from Form 4 and planned insider sales from Form 144.

Read the API reference OpenAPI schema llms.txt

Why this dataset?

  • Public SEC data, but still annoyingly messy in raw XML.
  • High-signal for AI agents monitoring insider activity and planned sell pressure.
  • Citation-backed payloads so an agent can verify the raw filing before acting.

What the API returns

  • Issuer name, ticker, CIK
  • Reporter or seller identity
  • Role and relationship to issuer
  • Normalized transactions or planned sale details
  • Direct SEC citation URLs
